# 质量评估评分标准文档

本文档详细说明系统中提示词质量评估和图片生成质量评估的评分标准与维度。

---

## 一、提示词质量评估

### 评估对象
**优先评估 DeepSeek 扩充后的提示词**（`enhanced_prompt`）。若未提供扩充结果，则评估原始提示词。

### 评分维度

#### 1. 信息密度 (information_density)
- **含义**：提示词中关键词的数量、多样性和独特性
- **评分逻辑**：
  - 基于逗号分隔的 token 数量统计
  - 统计覆盖的类别数（风格/光线/构图/质量）
  - 放宽标准：DeepSeek 扩充后的提示词通常较长，降低密度要求
- **分数范围**：0-100
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥75 | 丰富 |
  | ≥55 | 良好 |
  | ≥35 | 一般 |
  | <35 | 稀疏 |

#### 2. 结构完整性 (structure)
- **含义**：提示词是否包含完整的绘画要素
- **检查维度**：
  - 主体描述（人物/场景/物体）
  - 艺术风格（油画/水彩/二次元等）
  - 光线氛围（自然光/柔和光/逆光等）
  - 构图视角（特写/全景/鸟瞰等）
  - 色彩描述（红色/暖色调/鲜艳等）
  - 质感细节（纹理/光滑/金属等）
- **评分逻辑**：覆盖 3 个维度即可得高分
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥75 | 完整 |
  | ≥55 | 良好 |
  | ≥35 | 一般 |
  | <35 | 缺失 |

#### 3. 语言纯净度 (language_purity)
- **含义**：检测中英文混杂程度
- **评分逻辑**：
  - 纯中文或纯英文：100 分
  - 中英混合：根据混合程度扣分（放宽阈值）
  - DeepSeek 输出通常为纯英文，给予满分
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥85 | 纯净 |
  | ≥60 | 轻微混杂 |
  | ≥40 | 混杂 |
  | <40 | 严重混杂 |

#### 4. 具体性 (specificity)
- **含义**：描述的具体程度，避免模糊词汇
- **评分逻辑**：
  - 检测模糊词汇（"东西"、"物品"、"nice"、"good" 等）
  - 检测具体性指示词（"detailed"、"精确的" 等）
  - 放宽：降低模糊词惩罚力度
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥75 | 具体 |
  | ≥55 | 较具体 |
  | ≥35 | 一般 |
  | <35 | 模糊 |

#### 5. 风格完整性 (style_completeness)
- **含义**：是否包含完整的艺术风格描述
- **检查类别**：
  - 绘画风格（油画/水彩/素描等）
  - 艺术流派（印象派/抽象/写实等）
  - 渲染风格（3D/卡通/动漫等）
- **评分逻辑**：有 1 个类别即可得基础分
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥75 | 完整 |
  | ≥55 | 良好 |
  | ≥35 | 一般 |
  | <35 | 缺失 |

#### 6. 长度合理性 (length_appropriateness)
- **含义**：提示词长度是否适配 CLIP 模型的输入限制
- **评分逻辑**：
  - 最佳区间：**20-80 字**（适配 CLIP 约 70 字限制）
  - 低于 20 字：给予基础分 50 分
  - 20-80 字：100 分
  - 超过 80 字：适当扣分（放宽标准）
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥85 | 合适 |
  | ≥60 | 稍长/短 |
  | ≥40 | 偏长/短 |
  | <40 | 不合适 |

#### 7. 质量增强 (quality_boost)
- **含义**：质量关键词的使用合理性
- **检测词汇**："high quality"、"masterpiece"、"8k"、"精细" 等
- **评分逻辑**：
  - 无质量词：60 分（基础分提高）
  - 1-3 个：75-99 分
  - 4-6 个：100 分
  - 超过 6 个：适当扣分
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥85 | 优秀 |
  | ≥65 | 良好 |
  | ≥45 | 一般 |
  | <45 | 不足 |

#### 8. 增强效果 (enhancement_effect)
- **含义**：DeepSeek 扩充前后的质量提升程度
- **评分逻辑**：比较扩充前后在信息密度、结构完整性、具体性、风格完整性上的提升
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥85 | 显著提升 |
  | ≥65 | 有效提升 |
  | ≥45 | 轻微提升 |
  | <45 | 效果有限 |

### 综合评分权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 结构完整性 | 30% | 最重要 |
| 信息密度 | 15% | 降低权重 |
| 具体性 | 15% | |
| 风格完整性 | 15% | |
| 语言纯净度 | 10% | |
| 质量增强 | 10% | 提高权重 |
| 长度合理性 | 5% | 降低权重 |

### 综合等级

| 分数 | 等级 |
|------|------|
| ≥80 | 优秀 |
| ≥65 | 良好 |
| ≥50 | 一般 |
| ≥35 | 较差 |
| <35 | 很差 |

## 二、图片生成质量评估

### 评分维度

#### 1. CLIP 语义一致性 (clip_semantic)
- **含义**：生成图像与提示词的语义匹配程度
- **评分逻辑**：
  - 使用 CLIP 模型计算图像与文本的余弦相似度
  - 原始相似度通常较低（0.2-0.3 左右）
  - 使用 sigmoid 映射：让 0.2-0.3 的相似度也能得到中等分数
  - 映射公式：`score = 1 / (1 + exp(-10 * (similarity - 0.22)))`
- **分数范围**：0-1（映射后）
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥0.70 | 优秀 |
  | ≥0.50 | 良好 |
  | ≥0.30 | 一般 |
  | <0.30 | 较差 |

#### 2. 美学评分 (aesthetic)
- **含义**：图像的美学质量预测
- **评分逻辑**：
  - 基于 CLIP 图像特征的统计分布
  - 特征均值 + 标准差映射到 0-10 分
  - 放宽：基础分从 5.0 提高到 6.0，最低分从 1.0 提高到 3.0
  - 公式：`score = 6.0 + (feature_mean * 2.0) + (feature_std * 1.2)`
- **分数范围**：3.0-10.0
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥8.0 | 优秀 |
  | ≥6.0 | 良好 |
  | ≥4.5 | 一般 |
  | ≥3.0 | 较差 |
  | <3.0 | 很差 |

#### 3. 清晰度 (sharpness)
- **含义**：图像的清晰度和边缘锐利度
- **评分逻辑**：基于拉普拉斯算子的方差
- **分数范围**：0-100
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥70 | 优秀 |
  | ≥40 | 良好 |
  | ≥20 | 一般 |
  | <20 | 模糊 |

#### 4. 色彩丰富度 (color_richness)
- **含义**：色彩多样性和饱和度分布
- **评分逻辑**：基于 HSV 色彩空间的直方图熵值
- **分数范围**：0-100
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥80 | 丰富 |
  | ≥60 | 良好 |
  | ≥40 | 一般 |
  | <40 | 单调 |

#### 5. 构图平衡 (composition)
- **含义**：图像构图平衡性和视觉重心分布
- **评分逻辑**：
  - 计算图像重心与几何中心的偏差
  - 检测三分法构图
- **分数范围**：0-100
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥80 | 优秀 |
  | ≥60 | 良好 |
  | ≥40 | 一般 |
  | <40 | 失衡 |

#### 6. 噪声水平 (noise_level)
- **含义**：图像噪声程度（越高表示噪声越少）
- **评分逻辑**：基于 FFT 频域分析，高频能量比例
- **分数范围**：0-100
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥80 | 干净 |
  | ≥60 | 轻微 |
  | ≥40 | 一般 |
  | <40 | 噪点明显 |

#### 7. 对比度 (contrast)
- **含义**：图像明暗对比度和动态范围
- **评分逻辑**：基于灰度直方图的标准差和 Michelson 对比度
- **分数范围**：0-100
- **等级划分**：
  | 分数 | 等级 |
  |------|------|
  | ≥80 | 优秀 |
  | ≥60 | 良好 |
  | ≥40 | 一般 |
  | <40 | 平淡 |

### 综合评分权重

| 维度 | 权重 |
|------|------|
| 美学评分 | 25% |
| 清晰度 | 20% |
| 色彩丰富度 | 15% |
| 构图平衡 | 15% |
| 对比度 | 15% |
| 噪声水平 | 10% |

**注**：如果有 CLIP 语义一致性评分，会额外加入 30% 权重重新计算综合分。

### 综合等级

| 分数 | 等级 |
|------|------|
| ≥85 | 优秀 |
| ≥70 | 良好 |
| ≥55 | 一般 |
| ≥40 | 较差 |
| <40 | 很差 |

## 三、设计原则

### 提示词评分
1. **适配 DeepSeek 扩充**：优先对扩充后的提示词评分，认可 AI 扩充的价值
2. **适配 CLIP 限制**：长度合理性考虑 CLIP 约 70 字的输入限制
3. **放宽标准**：避免过于苛刻导致分数偏低，影响用户体验
4. **结构优先**：结构完整性权重最高，确保提示词包含必要的绘画要素

### 图片质量评分
1. **语义一致性放宽**：CLIP 原始相似度偏低，通过 sigmoid 映射让合理范围得到中等分数
2. **美学评分放宽**：提高基础分，避免生成图像美学分数过低
3. **多维度综合**：不依赖单一指标，综合评估图像质量
